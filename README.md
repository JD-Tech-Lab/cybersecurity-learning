## About the Room
This room explains how websites work. It focuses on how a browser sends a request to a server and how the server responds by sending data back to the browser.

## What I Learned
- A browser sends a request to a server through the internet.
- The server processes the request.
- The server sends a response back to the browser.
- Websites use this request and response process to show content.
